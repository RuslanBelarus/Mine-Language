# Mine-Language
Данный компилятор создан для компиляции .mc файла в папку script (datapack). Он завязан на локальной и вствроенной модификациях. Создан для упрощения создания карт и режимов, тем более для людей, незнакомых с коммандами. Строится на небольшом встроенном пакете Studio и ваших собственных модификацциях.
