# Mine-Language
Данный компилятор создан для компиляции .mc файла в папку script (datapack). Он завязан на локальной и вствроенной модификациях. Создан для упрощения создания карт и режимов, тем более для людей, незнакомых с коммандами. Строится на небольшом встроенном пакете *Studio* и ваших собственных модификацциях.
# Скриптинг
Язык расчитан на конвертацию строк в комманды. Язык состоит из глобальных и локальных функций. Глобальные функции конвертируюится в комманды и разделяются по функциям. Всего есть 2 основных функций: main *(выпролняется каждый тик)* и setup *(выполняется 1 раз при заходе в мир, либо прописования /reload)*. Так же можно создавать свои функции. Локальные функции ориентированы на создание и редактирование локальных перемнных и префабов.
**Для начала нужно импортировать** *Studio.mh*
> pack Studio.mh
Комманда будет добавлять в среду и компилятор определнный набор токенов и локализаторов для обработки базовых функций.
**После нужно указать функцию, в которой мы будем вести работу. Пускай это будет setup** *(выполняется 1 раз при входе или перезагрузке)*
Внутри функции мы будем писать комманды, выполнение которых мы ожидаем. Пускай мы будем выводить текст "*Привет, Мир!*". Для этого в функцию пишем следующее
*В этой строчке пробел заменяется троеточием, т. к. строка кода делится на токены по пробелам, и что бы оно не сломалось, мы заменяем на троеточие, которое после обработки превратится в пробел*
**Теперб выполним данный скрипт, через определенную комманду**
``` cmd
mlang script.mc C:\Users\admin\AppData\Roaming\.minecraft\saves\MineScript
```
``` mc
pack Studio.mh
fn setup
output "Привет,...Мир!"
end
```
